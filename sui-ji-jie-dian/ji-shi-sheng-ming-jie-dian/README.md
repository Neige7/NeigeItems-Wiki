# 即时声明节点

{% hint style="warning" %}
即时声明节点将增加物品生成的耗能，请尽量避免使用。
{% endhint %}

## 格式

<节点ID::节点类型::参数1\_参数2\_参数3...>

其中节点ID为非必要配置项

不指明节点ID的即时声明节点将无法二次调用
